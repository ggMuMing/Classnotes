11.9      学习内容：css选择、css3选择器、css继承性和层叠性、盒模型
1、CSS选择器
- 交集选择器： span.color { } 含义：class为color的span
- 后代选择器： div p{ } 含义：div元素下的p
- 组合（并集选择器）：h1,h2,h3,h4
2、CSS3选择器
- 儿子选择器：div>p { } 含义：选择div的直系p元素，下属div下的p不会被选中
- 下一个兄弟选择器:h1+p { } 含义：选择h1紧邻的下一个兄弟元素p
- 序选择器：li：first-child  last-child  li：nth-child（2）nth-child（2n+1）奇数表达式，2n+2偶数表达式，n从0开始
3、CSS继承性
父代已设置某些属性，子代会继承这些样式
注意：color、text-、line-、font-可以被继承，盒子、定位、布局不能被继承
4、CSS层叠性
CSS处理冲突的能力。多个选择器描述了同一个属性，使用属性取决于：
-	当该元素被选中时，要计算权重：id的数量、类的数量、标签数量
-	权重无进位
-	如果权重一样，采取就近原则
-	如果不能直接选中某个元素，通过继承性的话，权重为零
-	如果都没有选中目标元素，描述最近的有效，采取就近原则
5、权重深入!important
!important重要的三点：
-	!important提升的是一个属性，而不是一个选择器
-	!important无法提升继承的权重，该是0还是0
-	!important不影响就近原则，即权重相同，或都为0时，采取就近原则时，!important无效
6、CSS样式优先级
行内式>内部式>外链式
单个选择器优先级：！important>行内式>ID选择器>class选择器>元素
